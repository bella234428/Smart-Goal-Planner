# *SMART GOAL PLANNER*


## *Description*

This project provides a back-end solution for planning and managing goals. It includes functionalities for:

* *Planning goals*
* *Managing goal*

## *Project Setup*

The three core features of this Smart Goal Planner are implemented using back-end code located within a single folder. The project is built using:

* *Programming Language*: JavaScript ,CSS and html
